## Caso de Estudio Técnico: Alke Wallet

### 1. Breve descripción de la actividad
Desarrollo integral de una aplicación web transaccional (billetera digital) aplicando el patrón arquitectónico MVT (Modelo-Vista-Template), desde el modelado de la base de datos relacional hasta el despliegue de la interfaz de usuario.

### 2. Desafío principal que implicaba
El mayor reto fue garantizar la integridad de los datos financieros y la seguridad del acceso. Esto requirió sincronizar correctamente esquemas de bases de datos mediante migraciones estrictas, manejar relaciones complejas (uno a muchos, muchos a muchos) y proteger las operaciones de manipulación de datos (CRUD) contra accesos no autorizados.

### 3. Solución propuesta
Se construyó un sistema robusto utilizando vistas basadas en clases para estandarizar las operaciones CRUD. Se integró un sistema de autenticación nativo para restringir el acceso y se implementaron tokens CSRF en todos los formularios para prevenir ataques de falsificación de peticiones. La lógica de datos se encapsuló a través de un ORM, asegurando transacciones seguras y escalables. 

### 4. Herramientas técnicas utilizadas
* **Backend:** Python, Django 6.0
* **Base de Datos:** MySQL / SQLite
* **Frontend:** HTML5, CSS3 
* **Control de Versiones:** Git & GitHub

### 5. Principales aprendizajes alcanzados
* Comprensión profunda del ORM para la ejecución de consultas y modelado de datos sin escribir SQL directo.
* Gestión segura de variables de entorno (contraseñas y credenciales de BD) para separar entornos de desarrollo y producción.
* Manejo del flujo de autenticación de usuarios (Login/Logout) y protección de rutas mediante Mixins (`LoginRequiredMixin`).

### 7. Habilidades técnicas aplicadas
* Modelado de bases de datos relacionales.
* Programación Orientada a Objetos (POO).
* Desarrollo backend y gestión de rutas dinámicas.

### 8. Justificación de elección para el portafolio
He seleccionado este proyecto porque demuestra mi capacidad para construir soluciones tecnológicas "End-to-End" (de principio a fin). Alke Wallet refleja no solo el dominio de un framework moderno como Django, sino también una visión crítica sobre la seguridad de los datos transaccionales, algo indispensable en ecosistemas Fintech, LegalTech y cualquier entorno corporativo de alta responsabilidad.
